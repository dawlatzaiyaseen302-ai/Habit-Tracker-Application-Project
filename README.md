# Habit-Tracker-Application-Project
This Application is programmed through Python, which essentially asks a user various questions about their habits and it gives a footprint on an individual could replace their bad habits with good habits.

## Features
- Greets the user by name
- Add new habits
- Set weekly targets
- Log daily completion
- View all habits
- Show percentage progress
- Save data to habits.txt

## How to Run
```bash
python habit_tracker.py
```
